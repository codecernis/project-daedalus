# Project Daedalus

My personal portfolio website. Named after the Greek craftsman who built the labyrinth — because building a website from scratch with zero prior CSS or JavaScript experience felt about as straightforward as escaping one.

## What Is This

A single-page portfolio site built with vanilla HTML, CSS, and JavaScript. No frameworks. No build tools. No npm install. Just three files and stubbornness.

I built this to have a place on the internet that isn't LinkedIn, and to actually learn web development instead of just listing it on my resume and hoping nobody asks follow-up questions.

## Features

- **Catppuccin Mocha / Latte theming** — dark and light mode using the Catppuccin palette because I have taste
- **Neural network background** — animated nodes with pulsing connections that react to your mouse
- **Typing animation** — because the hero section felt too quiet
- **Scroll-triggered reveals** — sections fade in as you scroll, like a presentation that respects your time
- **3D tilt on project cards** — hover over them and they follow your cursor
- **Text scramble effect** — project titles and tech tags scramble on hover then resolve
- **Progress bar** — thin bar at the top so you know how far down you are
- **Active nav highlighting** — current section lights up in the navbar
- **Back to top button** — appears when you scroll down, does what it says
- **Easter egg** — type a secret word and find out

## Tech Stack

- HTML
- CSS
- JavaScript
- JetBrains Mono
- Catppuccin colour palette
- GitHub Pages for hosting
- Zero dependencies

## Why "Daedalus"

I name my projects after Greek mythology. Project Argus is my trading algorithm. This is the portfolio that holds everything together — the labyrinth, if you will. Except hopefully people can find their way around this one.
