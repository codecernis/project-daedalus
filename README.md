# Project Daedalus

My personal portfolio website. Named after the Greek craftsman who built the labyrinth — because building a website from scratch with zero prior CSS or JavaScript experience felt about as straightforward as escaping one.

## What Is This

A single-page portfolio site built with vanilla HTML, CSS, and JavaScript. No frameworks. No build tools. No npm install. No Stack Overflow tabs open (okay, maybe one). Just three files and stubbornness.

I built this to have a place on the internet that isn't LinkedIn, and to actually learn web development instead of just listing it on my resume and hoping nobody asks follow-up questions.

## Features

**The Essentials**
- Catppuccin Mocha & Latte theming — dark and light mode toggle because I have taste and indecisiveness
- JetBrains Mono everywhere because monospace fonts make everything look like you know what you're doing
- Fully responsive — looks good on desktop, tablet, and whatever phone you're doom-scrolling on
- Frosted glass navbar that shrinks when you scroll, because subtlety is a personality trait

**The Interactive Bits**
- Neural network particle background that reacts to your mouse — floating across the entire site like ambient intelligence
- Solar system experience section — each job is a planet orbiting a sun called "Data," because my entire career revolves around it (literally)
- A spaceship that flies between planets when you click them, complete with engine flames and a trail
- PETRONAS and UOB get orbit rings and moons because they were the real gigs, not internships
- 3D tilt effect on project cards and experience card — they follow your cursor like they're watching you
- Text scramble effect — hover over project titles and tech tags and watch them decode like classified files
- Typing animation on the hero tagline because the page felt too quiet without it

**The Polish**
- Scroll-triggered reveal animations — sections fade in as you reach them instead of all loading at once
- Active nav highlighting — the current section lights up so you know where you are
- Scroll progress bar — thin line at the top showing how far you've scrolled
- Back to top button — appears when you scroll down, does exactly what you think
- Brand-accurate planet colors — mSix, MISHU, PETRONAS, and UOB are their actual company colors
- Experience card border changes color to match whichever planet you've selected

**The Secret**
- Type a certain word on your keyboard and find out. I'm not telling you what it is. That defeats the purpose.

## What I Learned

This was my first real web project. I went in knowing what `<h1>` does and came out knowing what `requestAnimationFrame`, `IntersectionObserver`, `canvas.getContext('2d')`, CSS Grid, Flexbox, media queries, and `Math.atan2` do. I also learned that `z-index` is a lifestyle, not a property.

Every line was written by hand with guidance from Claude. No templates were copied. No code was generated and blindly pasted. If something looks wrong, it's my fault and I probably already know about it.
